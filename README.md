# Golden Ratio Calculator

A simple desktop utility for calculating Golden Ratio proportions quickly and accurately.

Designed for designers, architects, photographers and anyone working with balanced proportions.

---

## ✨ Features

- Instant Golden Ratio calculations (φ ≈ 1.618)
- Clean and minimal interface
- Fast numeric input
- No installation required
- Lightweight executable
